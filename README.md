# Workshop Attendance App

Ứng dụng chấm công nằm trong thư mục [du-an-moi](./du-an-moi).

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/tanchivas/8-3-for-U)

## Triển khai nhanh

1. Mở repo này trên GitHub
2. Bấm nút `Deploy to Render`
3. Đăng nhập Render và xác nhận tạo service
4. Sau khi Render deploy xong, bạn sẽ có một link công khai để mở web
